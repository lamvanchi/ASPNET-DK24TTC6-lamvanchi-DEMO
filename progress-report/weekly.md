## 2. Tiến độ thực hiện

### Tuần 1: Khởi tạo dự án

**Công việc đã thực hiện:**

* Tạo project ASP.NET Core MVC
* Cài đặt Entity Framework Core và SQLite
* Thiết kế cấu trúc thư mục (Controllers, Models, Views)
* Tạo database ban đầu
* Thêm dữ liệu mẫu

**Kết quả đạt được:**

* Project chạy thành công
* Hiển thị được danh sách sản phẩm

**Khó khăn:**

* Lỗi kết nối database
* Chưa quen với Entity Framework

**Hướng giải quyết:**

* Kiểm tra lại cấu hình connection string
* Tham khảo tài liệu và sửa lỗi

### Tuần 2: Xây dựng chức năng cơ bản

**Công việc đã thực hiện:**

* Xây dựng trang chủ
* Hiển thị danh sách sản phẩm
* Xây dựng trang chi tiết sản phẩm
* Thiết kế giao diện bằng Bootstrap

**Kết quả đạt được:**

* Website hiển thị sản phẩm đầy đủ

**Khó khăn:**

* Lỗi bố cục giao diện

**Hướng giải quyết:**

* Điều chỉnh CSS và sử dụng grid system của Bootstrap

### Tuần 3: Chức năng đăng nhập và giỏ hàng

**Công việc đã thực hiện:**

* Tích hợp ASP.NET Identity
* Xây dựng chức năng đăng ký, đăng nhập
* Xây dựng giỏ hàng

**Kết quả đạt được:**

**Hướng giải quyết:**
